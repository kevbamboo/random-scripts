# random-scripts

Random ideas, made mostly with ChatGPT and Claude

prosper:
Get listings from the prosper (p2p lending platform) API, use streamlit to show table and choose which ones you are interested in bidding

timetracker-email:
Extension that logs time you spend on each website, then will, at a specified time, open up your email and compose a summary of minutes spent on each site for you to send to someone else
